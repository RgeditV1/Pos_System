# Pos System

Proyecto de punto de venta hecho para aprender, practicar y crecer como desarrollador.

## Estado del proyecto

Actualmente esta en desarrollo.

## Objetivo

Este proyecto esta dedicado a toda persona que quiera aprender a usar y construir un punto de ventas, igual que yo.

## Contribuciones

Se puede contribuir libremente:

- proponiendo mejoras
- reportando errores
- enviando pull requests
- haciendo forks

## Uso comercial y forks

Puedes hacer forks, adaptarlo e incluso vender tus versiones derivadas, siempre y cuando respetes:

- la licencia del proyecto
- la autoria original
- las condiciones de GPL v3 al redistribuir (mantener la misma licencia en derivados y entregar acceso al codigo fuente)
